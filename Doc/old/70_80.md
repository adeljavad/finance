۷۲) **ثبت نمونه‌گیری (Sampling) متعارف بر اساس جدول آماری**  
en: `statistical_sampling`  
out: List[doc_id]  
code:  
```python
import random
pop = DocumentHeader.objects.filter(period=p)
n = min(62, int(pop.count()**0.5))  # ISA-530 متعارف
sample = random.sample(list(pop.values_list('id', flat=True)), n)
```

۷۳) **محاسبه «دقت نمونه» (Precision & Reliability)**  
en: `sample_precision`  
out: dict{precision, reliability}  
code:  
```python
exceptions = WorkPaper.objects.get(test_name='xx').exceptions
precision = exceptions / len(sample) if sample else 0
reliability = 1 - precision
```

### بعدی چی؟  
هرکدام از این ۱۰ تسک را که خواستی:
- تبدیلش می‌کنیم به یک `management command` (python manage.py audit_seal)  
 یا  
- یک Celery-Task با نتیجه‌ای که در مدل `AuditResult` ذخیره می‌شود  
 یا  
- یک API با DRF که JSON خروجی را می‌دهد و در Dashboard لنگ‌چین نمایش می‌دهی.

فقط بگو «تسک ۷۵ رو بزن» یا «Time-Tracking رو می‌خوام توی Celery»؛ بلافاصله کد کامل + تستش رو می‌سازیم.
