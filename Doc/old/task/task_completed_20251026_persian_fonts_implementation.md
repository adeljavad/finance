**تاریخ:** 2025-10-26

## شرح نیاز
پروژه نیاز به استفاده از فونت‌های فارسی استاندارد و زیبا برای بهبود خوانایی و زیبایی رابط کاربری داشت.

## راه‌حل پیاده‌سازی شده

### 1. انتخاب فونت Vazirmatn
- **فونت متن باز و رایگان:** Vazirmatn از Google Fonts
- **پشتیبانی کامل:** پشتیبانی از تمام وزن‌ها (100 تا 900)
- **بهینه‌سازی شده:** بهینه‌شده برای نمایش فارسی
- **سازگاری:** سازگار با تمام مرورگرها

### 2. پیاده‌سازی در صفحه آپلود
```html
<!-- فونت‌های فارسی -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Vazirmatn:wght@100;200;300;400;500;600;700;800;900&display=swap" rel="stylesheet">

<style>
    body {
        font-family: 'Vazirmatn', 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
        line-height: 1.6;
    }
</style>
```

### 3. پیاده‌سازی در صفحه پیش‌نمایش
```html
{% block extra_head %}
<!-- فونت‌های فارسی -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Vazirmatn:wght@100;200;300;400;500;600;700;800;900&display=swap" rel="stylesheet">
<style>
    body {
        font-family: 'Vazirmatn', 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
        line-height: 1.6;
    }
    .table th, .table td {
        font-family: 'Vazirmatn', 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    }
</style>
{% endblock %}
```

## نکات فنی
1. **Preconnect:** برای بهبود سرعت بارگذاری فونت
2. **Fallback fonts:** فونت‌های پشتیبان برای مرورگرهای قدیمی
3. **Line-height:** تنظیم مناسب برای خوانایی بهتر فارسی
4. **CDN:** استفاده از Google Fonts CDN برای سرعت و قابلیت اطمینان
