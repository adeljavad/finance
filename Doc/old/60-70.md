۶۹) **اسناد با توصیف‌های مشابه (Description Similarity > ۹۰٪)**  
en: `description_similarity`  
out: List[(doc_id, doc_id, score)]  
code:  
```python
from difflib import SequenceMatcher

headers = DocumentHeader.objects.filter(period=period)
pairs = []
for i, a in enumerate(headers):
    for b in headers[i+1:]:
        score = SequenceMatcher(None, a.description, b.description).ratio()
        if score > 0.9:
            pairs.append((a.id, b.id, score))
```

۷۰) **ثبت امضاء دیجیتال و هش SHA-256 روی هر سند (Tamper-Seal)**  
en: `digital_tamper_seal`  
out: Bool (success)  
code:  
```python
import hashlib, json

def seal_document(dh):
    payload = json.dumps([
        dh.document_number, str(dh.total_debit), str(dh.total_credit),
        dh.document_date, dh.description
    ], ensure_ascii=False).encode()
    dh.hash_sha256 = hashlib.sha256(payload).hexdigest()
    dh.is_sealed = True
    dh.save(update_fields=['hash_sha256', 'is_sealed'])
    return True
```

### بعدی چی؟  
هرکدام از این ۱۰ تسک را که خواستی:
- تبدیلش می‌کنیم به یک `management command` (python manage.py audit_seal)  
 یا  
- یک Celery-Task با نتیجه‌ای که در مدل `AuditResult` ذخیره می‌شود  
 یا  
- یک API با DRF که JSON خروجی را می‌دهد و در Dashboard لنگ‌چین نمایش می‌دهی.

فقط بگو «تسک ۶۴ رو بزن» یا «Sanction Hit رو می‌خوام توی Celery»؛ بلافاصله کد کامل + تستش رو می‌سازیم.
